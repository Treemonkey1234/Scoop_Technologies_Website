2. SMS and Phone Communication Consent

By providing your mobile phone number to Scoop Technologies, you are providing express written consent to receive text messages from us. This includes:

How We Obtain Consent

We obtain your consent through our web forms where you:

  • Enter your mobile phone number
  • Check the consent checkbox acknowledging you agree to receive SMS messages
  • Submit the form, confirming your consent

Types of SMS Messages

  • Transactional Messages: Account verification, password resets, security alerts

Opt-Out Procedures

You can revoke your consent and stop receiving SMS messages at any time by:

  • Replying "STOP" to any SMS message from us

12. SMS Opt-In Workflow Documentation

For Twilio Verification Compliance

This section documents our SMS opt-in process for compliance with carrier and regulatory requirements:

Opt-In Process:

  1. User visits our website and navigates to account creation or phone verification page
  2. User enters their mobile phone number in the designated field
  3. User sees clear language explaining they will receive SMS messages
  4. User checks a checkbox confirming consent to receive SMS messages
  5. User clicks submit, providing express written consent
  6. System sends verification code via SMS
  7. User enters code to complete verification

Consent Language Used:

"By providing your mobile phone number and checking this box, you consent to receive text messages from Scoop Technologies for account verification and service notifications. Message and data rates may apply. Reply STOP to opt out at any time."

Opt-Out Process:

  • Users can reply "STOP" to any SMS message
  • Users can update preferences in account settings
  • Users can contact support to be removed from SMS communications
